----------------INSTALLATION----------------
--------------------------------------------

   1.Extract

   2.Run the script install.bash. It will      
   require you to insert the password for
         administrator priviledges.
   
   3.The lns executable will be installed
   into /usr/bin/ folder, thus already in
   the path. A man entry will be added,you 
   will be able to access it through the
   command "man lns" to get an overview
      of the command and the language.
